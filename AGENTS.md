# Agent Information

## Development Server

- **Port**: 8888
- **Command**: `bun run dev`
- **URL**: [http://localhost:8888](http://localhost:8888)

## Features

- **Port Configuration**: The `dev` script in `package.json` is configured to run on port 8888 using `next dev -p 8888`.
- **Package Manager**: Bun is the preferred package manager.
